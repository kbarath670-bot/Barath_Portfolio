# Barath Kumar — DevOps Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript.  
Live at: **https://kbarath670-bot.github.io/portfolio/**

## Stack

- Pure HTML5 / CSS3 / JavaScript (no frameworks, no build tools)
- YouTube iframe for video hero background
- Hosted on GitHub Pages

## Sections

- **Hero** — Video background with typing animation
- **About** — Terminal-style bio widget
- **Skills** — Cloud, IaC, Containers, Linux, Observability, Security
- **Projects** — Server Health Monitor (featured)
- **Contact** — Email, LinkedIn, GitHub

## Deploy

Push to GitHub and enable Pages from the `main` branch root in repo Settings → Pages.
