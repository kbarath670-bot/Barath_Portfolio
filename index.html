AWS · Linux · Flask · Python · CloudWatch

Server Health Monitor

A real-time server health monitoring dashboard built on AWS, reading live metrics directly from the Linux /proc filesystem. The system surfaces CPU usage, memory consumption, disk I/O, and uptime into a web dashboard backed by a Flask API — with CloudWatch Agent integration for metric shipping and SNS alerting for threshold breaches.

  • Reads from /proc/stat, /proc/meminfo, /proc/diskstats for raw system data
  • Flask REST API serves metrics to a responsive front-end dashboard
  • CloudWatch Agent configured to ship custom metrics; SNS triggers on breach
  • Deployed on a custom VPC with public/private subnets across two AZs
  • HTTPS secured via Apache mod_ssl; session-authenticated admin interface
