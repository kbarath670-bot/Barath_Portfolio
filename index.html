Behind the Product

I'm currently a Server Support Analyst at eClerx, managing cloud and on-premises server infrastructure, system monitoring, and infrastructure troubleshooting. Previously, as an IT Support Technician at I Digimeta Technology, I administered Windows Server environments, Active Directory, and virtualization infrastructure for 100+ end-users — the operational foundation that informs how I approach automation today.

I'm building toward DevOps and Cloud Engineering through hands-on, self-directed work: AWS fundamentals (EC2, VPC, CloudWatch), Infrastructure as Code with Terraform and Ansible, containerization with Docker and Kubernetes, and CI/CD pipelines using GitHub Actions and Jenkins. I built and deployed a real-time Server Health Monitor on AWS with CloudWatch integration to put these skills into practice.

I hold a B.Sc. in IT from Karpagam Academy, a CISSP from (ISC)², and an Oracle Certified Expert — Database SQL. My goal is a DevOps Engineer or Cloud Engineer role where I can grow this automation toolkit on real production systems.
